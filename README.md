What is EngPL and what makes it different from c++ plython c# and more

EngPL is a beginner-friendly, English-like programming language designed
to help people create logic, games, and creative programs without
complex syntax.

This project is currently in **Beta 0.02.x**.

## Features
- English-style syntax
- Variables and math
- Conditions (`when`)
- Loops (`repeat`)
- Music support (`music(a, b#, c)`)
- Built-in editor mode
- Help system

## Example
```text
write(Hello)

var(a)
a(3)

repeat(a)
write(loop)
end

music(c, d, e)
